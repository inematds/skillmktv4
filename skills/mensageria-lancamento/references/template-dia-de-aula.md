# Template de Dia de Aula: Dois Modelos Completos

Perguntar ao usuário qual modelo usar. Se horário entre 6h-12h: modelo manhã. Se 17h-22h: modelo noite.

## Aulas 2 a 5 (modelo manhã, com gravação do dia anterior)

Mesma estrutura da Aula 1, com uma diferença: a gravação do dia anterior NÃO aparece de manhã (já foi liberada às 12h do dia anterior). A primeira mensagem é direto a preparação às 6h45.

Blocos especiais:
- **Aula 4 (quinta):** Após ao vivo, FICHA DE INTERESSE + gravação às 9h. API ficha às 12h.
- **Aula 5 (sexta):** Gravação às 12h inclui reforço da ficha de interesse.

## Blocos Especiais (ambos os modelos)

### Quinta (após Aula 4):
- Ficha de interesse (grupo), SEPARADO da antecipação
- Antecipação Aula 5: bloco SEPARADO com bullets completos
- Ver `references/ficha-de-interesse.md`

### Sexta (modelo noite):
- 8h: gravação Aula 4 + ficha de interesse EMBUTIDA no mesmo bloco
- 12h: dois blocos: 1) API ficha (com vídeo) 2) Preparação texto (sem vídeo)
- 21h30: antecipação com ficha no final

### Sexta (modelo manhã):
- 12h: gravação Aula 5 + reforço ficha no mesmo bloco
